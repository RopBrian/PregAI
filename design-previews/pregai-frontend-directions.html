Standalone browser preview. No React app code changed.

Clinic Companion Maternal Journal Clinical Command Home Care Diagnostic Atelier

Five complete visual languages for PregAI.

Each direction keeps the current product surfaces: landing, auth, terms, header, sidebar, chat and scan upload, education, scan history and report, profile and privacy, admin dashboard, and system logs.

Intent: fewer floating cards, more real human presence, more editorial composition, and copy that sounds like a caring product rather than a machine. The previews use browser-frame mockups so you can compare how each design would feel in use.
